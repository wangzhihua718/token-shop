# Token Shop 项目总结

## 项目概述

Token Shop 是一个基于BMAD方法论设计的Token售卖平台，采用前后端分离架构，支持Token的创建、展示、购买、管理和结算等完整业务流程。

## BMAD开发流程执行

### 1. 发现阶段 (Discovery)
- 完成市场需求分析
- 确定目标用户群体（买家、卖家、管理员）
- 明确核心价值主张

### 2. 规划阶段 (Planning)
- 编写产品需求文档 (PRD)
- 完成架构设计文档
- 创建用户故事卡片
- 定义数据模型和API规范

### 3. 实施阶段 (Implementation)
- 前端：Vue3 + Element Plus + TypeScript
- 后端：SpringBoot + Java8 + MyBatis-Plus
- 数据库：MySQL 5.7+
- 缓存：Redis

### 4. 交付阶段 (Delivery)
- 代码完成
- 数据库脚本准备
- API文档生成 (Swagger)

## 技术亮点

1. **前后端分离架构**：清晰的职责划分，便于维护和扩展
2. **RESTful API设计**：遵循REST规范，接口统一
3. **JWT认证**：无状态认证，支持分布式部署
4. **MyBatis-Plus**：简化数据库操作，提高开发效率
5. **Element Plus UI**：现代化的前端组件库
6. **TypeScript**：类型安全，提高代码质量

## 启动指南

### 环境要求
- JDK 8+
- Node.js 16+
- MySQL 5.7+
- Redis 5+
- Maven 3.6+

### 后端启动
```bash
cd backend
# 1. 初始化数据库
mysql -u root -p < src/main/resources/db/init.sql

# 2. 修改配置文件
# 编辑 src/main/resources/application.yml，配置数据库和Redis连接

# 3. 编译运行
mvn clean install
mvn spring-boot:run
```

### 前端启动
```bash
cd frontend
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev
```

### 访问地址
- 前端：http://localhost:5173
- 后端API：http://localhost:8080
- Swagger文档：http://localhost:8080/swagger-ui.html
